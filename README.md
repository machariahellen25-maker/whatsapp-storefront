# WhatsApp Storefront

A simple beauty booking page where services go straight to WhatsApp.


## 📱 Overview

This is a small frontend demo for a beauty booking page.

It shows how a salon can list services and let people book directly on WhatsApp without extra steps.


## 💡 Why it exists

Booking beauty services often takes too many messages and delays.

People ask, wait, and sometimes drop off.

This removes that gap.

You see a service, you see the price, and you book directly.


## 💄 What’s inside

- Home section  
- List of beauty services with prices  
- WhatsApp booking button for each service  
- Simple mobile-friendly layout  
- Founder section (Hellen)


## ⚙️ How it works

1. Pick a service  
2. Tap the WhatsApp button  
3. Send the message  
4. Agree on time  

That’s it.
